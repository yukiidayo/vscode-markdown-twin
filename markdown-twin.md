# Markdown Twin Preview

Markdown Twin lets your team review translated Markdown without leaving VSCode.

> Keep the original structure. Review the translated result. Share it faster.

## What you can preview

* **Bold text**, *italic text*, and `inline code`
* Links, quotes, lists, tables, and code blocks
* Translated Markdown source
* Bilingual view with original and translated text

## Review checklist

* [x] Headings are preserved
* [x] Lists keep their indentation
* [ ] Tables are still readable
* [ ] Code blocks are not translated

| Mode        | Purpose                        |
| ----------- | ------------------------------ |
| Translation | Read the translated document   |
| Bilingual   | Compare source and translation |
| Source      | Copy translated Markdown       |
